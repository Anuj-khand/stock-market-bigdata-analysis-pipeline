# Glue Table Schema

Table contains stock market events generated by Kafka Producer and stored in S3.

## Columns
- symbol (string)
- open (double)
- high (double)
- low (double)
- close (double)
- volume (int)
- event_time (string)

## Partition Columns
- year (string)
- month (string)
- day (string)

Partitioning improves Athena performance and reduces scan cost
